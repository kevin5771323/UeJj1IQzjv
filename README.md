## 前言

随着我国教育改革的不断深入，研学旅行作为一种新兴的教育形式，越来越受到家长和学校的关注。乡村研学旅行平台是基于小程序和Spring Boot技术，致力于为广大学生、家长和学校提供一个便捷、高效的研学旅行解决方案。

## 内容介绍

本项目主要包括以下功能模块：首页、研学课程、目的地推荐、个人中心等。用户可以通过平台浏览各类研学课程，了解课程详情，选择适合自己的研学目的地，并进行在线报名。此外，平台还提供个人中心功能，方便用户管理个人信息和订单。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpStudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12/14/16
